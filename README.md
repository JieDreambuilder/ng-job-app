# ng-job-app
一个基于angular、gulp、less的求职APP。

### 实现功能

* 登录、注册
* 浏览职位
* 查看职位详情
* 个人页面
* 投递简历
	* 根据简历状态筛选
* 收藏职位
* 筛选职位
	* 根据城市、薪资、公司规模筛选职位
### 项目信息

* 包管理工具：bower、npm
* 自动化构建工具：gulp
* 自适应方案：rem
* 数据：项目数据皆为json格式的假数据，存放地址为data目录下。

### 运行

* npm update
* gulp
